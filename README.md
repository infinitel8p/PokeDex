## Installation:

### Windows:
1. Go to the [Releases](https://github.com/infinitel8p/PokeDex/releases) page and download the latest `[...]setup.exe` or `[...]en-US.msi` file.
2. Run the installer and follow the instructions. You might need to check if your antivirus is blocking the installation.
3. Click on More Informatiton in the Microsoft Defender SmartScreen Popup and then on Run Anyway

### MacOS:
1. Go to the [Releases](https://github.com/infinitel8p/PokeDex/releases) page and download the latest `.dmg` file. Check the latest release for the correct file according to your architecture.  
2. Open the `.dmg` file and drag the app to your Applications folder. Right click the Applications folder and click on Open.
3. Locate the app in your Applications folder and control-click the app icon. Click Open in the context menu.   
   The app is saved as an exception to your security settings, and you can open it in the future by double-clicking it just as you can any registered app.
